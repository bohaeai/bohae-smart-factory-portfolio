# 제성 공정 자동화 — 도메인 지식 (SSOT)

> **문서 역할**: 제성 자동화 프로젝트의 센서/공정/유체역학/규제 도메인 지식 영구 보존  
> **최종 갱신**: 2026-03-24  
> **검수**: Gemini / GPT 교차 검증 완료

---

## 1. 제성(配成) 공정이란

제성은 주류 제조에서 **원료를 배합하여 완제품 액체를 만드는 공정**이다.

```
원재료 (주정, 정제수, 원액, 첨가물)
    ↓ 정량 투입
배합탱크에서 혼합/교반
    ↓ 품질 확인 (도수, 브릭스)
    ↓ 여과
버퍼탱크 → 충전기(필러) → 완제품
```

### 핵심 품질 지표

| 지표 | 단위 | 의미 | 측정 방법 |
|------|------|------|---------|
| **ABV (Alcohol By Volume)** | % | 알코올 도수 | 밀도 측정 → 환산 또는 증류법 |
| **Brix** | °Bx | 당도 (용해 고형분) | 굴절계 |
| **pH** | - | 산도 | pH미터 |
| **전도도** | µS/cm | 용액의 전기 전도성 | 전도도 센서 |

### 제품군별 차이

| | A형: 희석 주류 | B형: 과실 베이스 |
|---|---|---|
| 대표 | 잎새주, 보해골드, 호카이 | 복분자, 매취순 |
| 주 원료 | 주정 + 정제수 | **원액(과실)** + 주정 + 정제수 |
| 핵심 품질 | ABV (도수) | **Brix (당도) + ABV** |
| 원액 특성 | 없거나 극소량 | 고점도, 고Brix, 과육/당분 |
| 여과 부하 | 낮음 | **높음** (과육/찌꺼기) |
| CIP 강도 | 보통 | **강** (당분 잔류) |

## 2. 8개 단위공정(Zone) 상세

### Zone 1: 원료/정제수 저장

**목적**: 원료 탱크의 재고를 실시간으로 파악하고, 과충전/공전 사고를 방지한다.

| 센서 유형 | 왜 필요한가 | 선정 기준 |
|----------|-----------|---------|
| **연속 레벨 (Radar 80GHz)** | 재고 0~100%를 mm 단위로 연속 측정. "얼마 남았는지"를 숫자로 본다 | 초음파 대비: CIP 증기/거품에 강인, 80GHz는 빔이 좁아 탱크 내부 구조물 간섭 최소, 비접촉이라 위생 유지 |
| **포인트 레벨 HH (진동식 Vibronic)** | 과충전 시 유입밸브를 **하드웨어 인터록**으로 차단. 레이더가 고장나도 독립 동작 | SIL2 가능. 레이더와 독립된 안전 레이어 |
| **포인트 레벨 LL (진동식)** | 탱크 바닥 도달 시 이송 펌프 **공회전(Dry-run) 강제 정지** | 펌프 공전 시 임펠러 손상/수명 단축 방지 |

> **왜 초음파가 아니라 레이더인가?**  
> 초음파는 CIP 고온 증기, 교반기 거품/와류, 탱크 내부 구조물 반사, 결로/응축에 의해 오작동한다.  
> 80GHz 레이더는 이 모든 조건에서 안정적이다.

### Zone 2: 정량 투입

**목적**: 주정, 정제수, 원액이 배합탱크에 **정확히 얼마나** 들어가는지 측정한다.  
이 데이터가 없으면 배치별 원가 산출이 불가능하다.

| 센서 | 대상 유체 | 왜 이 센서인가 |
|------|---------|-------------|
| **Magmeter (전자식 유량계)** | 정제수 | 정제수는 전도도가 높아서(>5 µS/cm) Magmeter 사용 가능. Coriolis보다 저렴 |
| **Coriolis (질량유량계)** | 주정 | ① 주정 전도도가 낮을 수 있어 Magmeter 불가 가능성 ② 밀도 측정으로 ABV 직접 환산 ③ 질량 기반이므로 온도 변화 영향 없음 |
| **Magmeter 또는 Coriolis** | 원액 | 별도 배관이면 필요. 같은 배관이면 불필요. 원액이 드럼/호퍼면 탱크 계량(WIT) 대체 |
| **압력 센서** | 각 투입 펌프 토출부 | 밸브 미개방 상태에서 펌프 가동 시 **수격현상(Water Hammer)** 감지 → 배관 파열 방지 |

> **Magmeter vs Coriolis 선택 분기**:
> ```
> 유체 전도도 ≥ 5 µS/cm → Magmeter (저렴)
> 유체 전도도 < 5 µS/cm → Coriolis (비싸지만 전도도 무관)
> ```
> 이 1개 질문이 **300~800만원**의 비용 차이를 만든다.

### Zone 3: 소량 첨가물

**목적**: 수작업 유지하되 **기록을 자동화**한다. "누가 언제 몇 g 넣었는지" 전산에 남겨 HACCP 추적성을 확보한다.

| 센서 | 왜 필요 |
|------|--------|
| **로드셀 (Load Cell)** | 작업자가 첨가물을 올려놓으면 무게 차이(Loss-in-weight)를 MES에 자동 전송 |

> 용해조(Pre-mix tank)가 있다면 추가 제어 라인 필요 여부를 현장에서 확인한다.

### Zone 4: 배합/교반

**목적**: 배합 중 체적, 온도, 압력, 교반기 상태를 감시한다.  
특히 **TIT-401 (온도)**은 도수/브릭스 헌팅의 핵심 원인 변수이다.

| 센서 | 왜 필요 |
|------|--------|
| 레이더 LIT-401 | 배합 중 체적 감시. 교반기 와류+CIP 증기에서 초음파 불가 → 레이더 필수 |
| 진동식 LSH/LSL-401 | 배합액 넘침 방지 + 배출 완료 시 펌프 보호 |
| **Pt100 TIT-401** | **온도에 따라 밀도가 바뀌고, 밀도가 바뀌면 도수 측정값이 달라진다.** 헌팅 원인 추적의 핵심 |
| 압력 PIT-401 | 순환 펌프 이상/배관 막힘 조기 감지 |
| **AGIT-401** | 교반기 run/speed/fault/전류. 멈추면 배합 불균일 → 품질 문제 |

### Zone 4.5: 여과 (필터)

**목적**: 배합 후 버퍼/충전기로 넘기기 전, 과육·활성탄 가루를 걸러낸다.

| 센서 | 왜 필요 |
|------|--------|
| **PDIT-451 (차압 센서)** | 필터 전/후단 압력 차이가 기준 초과 시 **교체 경보**. 필터 막히면 펌프 과부하 → 터짐. 지금은 유량 떨어지는 걸 사람이 느끼고 교체 |

> 특히 복분자·매취순 등 과실주는 과육/당분 찌꺼기가 많아 필터 막힘 빈번.

### Zone 5: 품질 조정 + 샘플링

**목적**: 배합 중 **순환 루프**에서 실시간 ABV/Brix를 판정하고, 랩 검증용 샘플을 채취한다.

| 센서 | 왜 필요 |
|------|--------|
| **AIT-501 (인라인 분석기)** | 샘플링 없이 실시간 품질 판정. 현재 30분 랩 대기 → 0분으로 단축 |
| **SP-501 (위생형 샘플 포트)** | 센서 교정/검증용 랩 샘플 채취. **없으면 온라인 센서 신뢰도를 증명할 방법이 없다** |

> **AIT-501 제품군별 분기**:
> | 제품군 | 센서 | 측정 원리 |
> |-------|------|---------|
> | A형 (소주) | Coriolis → 밀도 → ABV | 밀도와 ABV는 알려진 상관관계 |
> | B형 (복분자) | 굴절계(Refractometer) → Brix | 굴절률과 당도는 직접 상관 |

### Zone 6: 이송/버퍼

**목적**: 배합 완료 후 충전 라인으로의 **최종 산출량**을 측정한다.

| 센서 | 왜 필요 |
|------|--------|
| FIT-601 (Magmeter) | 투입량(FIT-201+202+203) 대비 **수율** 산출. 손실 구간 특정 |
| LIT-601/LSH/LSL-601 | 버퍼 수위. 충전기에 끊김 없는 공급 |

> **수율 2단계 폐쇄**:
> ```
> 1차: 투입 합계 vs 배합 산출(FIT-601) = 탱크/배관/여과 손실
> 2차: 배합 산출 vs 충전 good count × 실제 충전량 = 충전/포장 손실
> ```

### Zone 7: CIP 세척

**목적**: 세척 완료를 **시간이 아닌 센서 데이터**로 자동 판정한다.

| 센서 | 왜 필요 |
|------|--------|
| **AIT-701 (전도도)** | 물↔산↔알칼리 전환(Phase Shift) 자동 감지. 전도도가 기준 이하 = 헹굼 완료 |
| **FIT-701 (CIP 리턴 유량)** | 세척 유속 1.5m/s 이상 난류 보증. 유속 부족 = 세척 불량 |
| **TIT-701 (온도)** | 살균 온도 85°C 도달 확인. 미달 = 재세척 |

> **향후 확장**: 당류/색도 제품은 전도도만으로 phase shift 못 잡을 수 있음.  
> AIT-702 (광학 탁도/interface sensor) 추가 여지를 배관 설계에 남겨둘 것.

## 3. 센서 선정 근거 — 기술적 이유

### 레이더 vs 초음파 (레벨 측정)

| | 초음파 | 80GHz 레이더 |
|---|---|---|
| CIP 고온 증기 | ❌ 오작동 | ✅ 영향 없음 |
| 교반기 거품/와류 | ❌ 난반사 | ✅ 80GHz 빔이 투과 |
| 탱크 내부 구조물 | ❌ 반사 간섭 | ✅ 좁은 빔각 |
| 결로/응축 | ❌ 센서면 부착 | ✅ 비접촉 |

### Coriolis vs Magmeter (유량 측정)

| | Magmeter | Coriolis |
|---|---|---|
| 전도도 요구 | ≥5 µS/cm 필수 | 전도도 무관 |
| 밀도 측정 | ❌ | ✅ → ABV 환산 |
| 측정 단위 | 체적 (L) | **질량 (kg)** |
| 온도 영향 | 체적 변화 있음 | 질량 불변 |
| 가격 | ~450~700만 | ~600~1,500만 |

### 클램프온 유량계 (POC용)

| 특성 | 내용 |
|------|------|
| 설치 | **배관 외부 부착** — 배관 절단 불필요 |
| 원리 | 초음파 전파 시간차 |
| 필수 조건 | **만관(Full-pipe)** — 배관 내부 100% 충만 필요 |
| 실패 조건 | 반관, 낙하 배관, 기포 |
| 최적 설치 | **수직 상승 직관부** (바닥→위) 30~50cm 이상 |
| 측정 단위 | L/min (체적). kg 아님. 질량은 밀도 보정 필요 |

## 4. 밸브 도메인 지식

### 밸브 유형 분류

| 유형 | 용도 | 특징 |
|------|------|------|
| **Single-seat** | 단순 on/off 차단 (투입, 이송) | 일반적. 저렴 |
| **Mixproof (Double-seat)** | product-CIP 교차점 | 교차오염 방지. double block-and-bleed 구조. 2~3배 비쌈 |

> **CV-401 (배합탱크 하부)과 CV-701 (CIP 리턴)**은 product와 CIP가 교차하므로 mixproof 검토 필수.

### 밸브 피드백

모든 자동밸브에 **open / close / fault** 신호를 PLC로 가져간다.

- "열라고 했는데 안 열렸다" → 생산 중단 원인 추적
- "닫혔다고 믿었는데 seat leak" → 교차오염 사고 방지
- Phase 3 자동제어에서 **필수**

### 수격현상(Water Hammer) 방지

펌프가 전속 주행 중 밸브를 갑자기 닫으면 **배관이 터진다**.

```
해결: VFD 인버터 다단 제어
  목표 0~90%  → 펌프 고속
  목표 90~98% → 펌프 저속 (Dribble flow)
  목표 98~100% → 극저속 + 밸브 서서히 닫기
  목표 도달    → 펌프 정지 + 밸브 완전 차단
```

**인버터(VFD)는 선택이 아니라 안전 필수 장비.**

## 5. 설치 기준 — 3-A / EHEDG

식품 위생 센서 설치 시 반드시 충족:

| 기준 | 내용 | 미충족 시 |
|------|------|---------|
| 자배수 (Self-draining) | 센서 포켓에 액체 고이지 않는 각도 | 잔류 → 오염원 |
| 데드레그 최소화 | 센서 노즐 돌출 ≤ 1.5D | 세척 사각지대 |
| CIP spray 도달 | spray ball 커버리지 확인 | 세척 불량 |
| 누설 검출 | 밸브 seat leak drain 확인 가능 | 교차오염 |
| 접액부 재질 | 316L / EPDM / PTFE | 식품 안전 |
| 표면 조도 | Ra ≤ 0.8 µm | 3-A 기준 |

## 6. 규제/법적 도메인

### 주세법 관봉

- 주정 밸브/계량기에 **세무서 납땜 관봉(씰)**이 있을 수 있음
- 임의 절단 = **주세법 위반**
- 배관 개조 전 관할 세무서 **사전 신고** 필요
- 유량계 설치 시 **KOLAS 검교정 성적서** 구비

### 방폭 구역

- 주정(에탄올) = 인화성 액체
- 주정 취급 구역이 **방폭 구역(Zone 1/2)**으로 지정되면:
  - 모든 센서: 본질안전(Ex ia) 또는 내압방폭(Ex d) 인증
  - 배선: 방폭 케이블/그랜드
  - 제어반: 방폭 판넬
  - **비용 1.5~2배 증가**

## 8. IO-Link / 통신 아키텍처

### Phase 0~2 (데이터 수집만)

```
센서 ──IO-Link──▶ IO-Link Master ──Modbus TCP──▶ Edge Server (N100)
                                                    │
                                                MQTT Broker ([REDACTED_BROKER_IP]:8883)
                                                    │
                                        bohae/v1/jeseong/FIT-201/flow
                                        bohae/v1/jeseong/LIT-401/level
                                                    │
                                                대시보드 / MES
```

### Phase 3 (자동 제어)

```
센서 ──IO-Link──▶ IO-Link Master ──동시──▶ Edge Server (모니터링)
                        │
                        ▼
                      PLC ──▶ 자동밸브 (CV-201~701)
                        │──▶ 인버터 VFD (펌프 속도)
                        │
                     제어 로직:
                     - FIT-201 적산 = 목표 → CV-201 닫기
                     - AIT-501 ABV = 목표 → 순환 정지 → 이송
                     - PDIT-451 차압 > 한계 → 필터 교체 알람
```
